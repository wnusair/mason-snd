Upload Roster

Create a new roster or update an existing one from an Excel file

How to use this feature

Download → Edit → Upload Workflow:

  1. Download an existing roster as Excel from the roster view page
  2. Edit competitor and judge names as needed
  3. Upload the modified file here (select "Update Existing Roster")
  4. The system will automatically match users and update the roster

What you can edit:

  • Names - Change competitor or judge names
  • Add/Remove rows - Add new competitors or judges, or delete rows
  • Reorder - Change the ranking order

How matching works:

  • User ID columns (shaded blue) are used for matching when present
  • If you change a name but keep the User ID, it finds the right person by ID
  • If you delete the User ID, it will try to match by the name
  • Event ID and other metadata is automatically updated

⚠️ Important: The "Rank View" sheet is the primary source for competitors. Make sure to edit this sheet for competitor changes.
