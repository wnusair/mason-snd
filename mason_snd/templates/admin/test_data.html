Create Test Users

Creates 15 test students (Student1-Student15) and their parents (Parent1-Parent15) with the specified password.

Enroll in Events

Makes all test students join 1-3 random events from available events.

Sign Up for Tournaments

Makes all test students sign up for 1-2 random tournaments with their parents as judges.

⚠️ Cleanup Test Data

This will permanently delete all test students, parents, and their related data (event enrollments, tournament signups, etc.). This action cannot be undone.

📋 Instructions

  1. Create Test Users: First, create the 15 test students and their parents. Set a password that you'll remember.
  2. Enroll in Events: Make sure you have some events created first, then enroll the test students in random events.
  3. Sign Up for Tournaments: Make sure you have tournaments created, then sign up students for tournaments with their parents as judges.
  4. Test Login: You can now log in as any test user using their email (student1@gmail.com, parent1@gmail.com, etc.) and the password you set.
  5. Cleanup: When you're done testing, use the cleanup function to remove all test data.

Test User Credentials:

  • Students: student1@gmail.com through student15@gmail.com
  • Parents: parent1@gmail.com through parent15@gmail.com
  • Password: Whatever you set above (default: testpass123)
